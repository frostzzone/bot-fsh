Api documentation

Add ?plain=1 at the end of urls to get non json responses
{} is the request type needed
[] for parameters (?thing=thang)

Fsh bot api

/ping {get} - Responds with bot ping

/info {get} - Responds with bot info

Text api

/filter {get} [text] - Responds with true if text doesn't have bad works else false

/censor {get} [text] - Responds with inputed text but bad words are censored

/page {get} [url] - Get page contents of an url

Image api

/animal {get} [animal] - Responds with random animal (leave animal empty for animal list)

/imagine {get} [text] - Responds with ai image from text (may take up to 2 minutes)

Deprecated

/fish {get} - get random fish image
